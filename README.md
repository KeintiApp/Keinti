# KeintiApp - Proyecto Reconstruido

Este proyecto ha sido reconstruido desde cero manteniendo solo las pantallas esenciales.

## Estructura

### Backend
- Servidor Express básico en Node.js
- Puerto: 3000
- Ubicación: `Backend/src/server.js`

### Frontend
- React Native app
- Pantallas mantenidas:
  - LoginScreen.tsx
  - RegisterScreen.tsx
  - ProfilePhotoEdit.tsx
  - FrontScreen.tsx

## Instalación

### Backend
```bash
cd Backend
npm install
npm start
```

### Frontend
```bash
cd Frontend
npm install
npm start
```
